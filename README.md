# graduation_project
这是一个毕设文件
其中有部分代码来源于博客，如有侵权请联系邮箱3313628054@qq.com 删除代码
目录：
        1 ICP_Quat
        2 ICP_SVD
        3 RotationTest
        4 kdtreeTest
文件1是ICP四元数的运行工程
文件2是ICP SVD的运行工程
文件3是三种旋转方法（欧拉角，旋转矩阵，四元数）相互转化的实验
文件4是KDTree的实验工程
其中文件3中含有暴力匹配和kdtree匹配的测试代码和对比代码

注意 
        这里采用的python版本是3.6 所用的库是numpy,scipy,matlpy
       代码以 *test.py格式都是可以直接运行的代码 在终端键入 python *test.py
